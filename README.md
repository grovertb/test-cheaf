# Test-cheaf

## Demo

La aplicación está desplegada en [https://test-cheaf-production.up.railway.app/](https://test-cheaf-production.up.railway.app/).

## Ejecución Local

### Pre-requisitos

- [npm](https://www.npmjs.com/) o [yarn](https://yarnpkg.com/)

### Instalación de Dependencias

```bash
npm install
# O
yarn
```

### Inicialización

```bash
npm run start
# O
yarn start
```
